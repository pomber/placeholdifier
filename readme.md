## NPM Package

You can use it in your site by importing the CSS from unpkg and adding the `placeholdify` class somewhere:

```html
<html>
  <head>
    <link
      href="https://unpkg.com/placeholdifier/placeholdifier.css"
      rel="stylesheet"
    />
  </head>
  <body>
    <h1>Hello World</h1>
    <h1 class="placeholdify">Hello World</h1>
  </body>
</html>
```
